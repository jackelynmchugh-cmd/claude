trader — Stock trader

                  pre-4S: alerts only

                  hacks linked servers for shorts

backdoor        → no args, scans full network and backdoors

autoroot        → no args, roots all reachable servers

orch            → no args, manages up to 2 batchers
                  auto-migrates to pserv-0 if available
                  picks best targets by score automatically

buyer — Server buyer
buyer           → no args, auto-buys and upgrades pservs
                  starts at 8GB, upgrades all to same tier

upgrade — Home upgrade
upgrade         → no args, buys home RAM and core upgrades
                  when you can afford them

contracts — Contract solver
contracts       → no args, scans all servers for coding
                  contracts and attempts to solve them

dnet            → no args, only works once darknetscape.exe
                  is unlocked, boots darknet connection

sell — Stock sell all
sell            → no args, emergency sell all stock positions
                  useful before an augmentation install reset

                    starts: autoroot, manual-hack, orchestrator,
                    server-buyer, stock-trader, tor-manager
                    runs backdoor → contracts → home-upgrade
